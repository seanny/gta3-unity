# Default Controls

## At All Times

### Camera Modes
PC: [C] or [HOME]
XBOX: [BACK]
PS2: [SELECT]

### Pause Menu
PC: [ESC]
XBOX: [START]
PS2: [START]

## On Foot

### First-Person Look
PC: [NUM LCK] + [8] / [4] / [2] / [6]
XBOX: [RIGHT STICK BUTTON]
PS2: [RIGHT STICK]

### Move
PC: [W] / [A] / [S] / [D] / [LEFT ARROW] / [UP ARROW] / [DOWN ARROW] / [RIGHT ARROW]
XBOX: [LEFT STICK]
PS2: [LEFT STICK]

### Jump
PC: [SPACE] / [CTRL]
XBOX: [X]
PS2: [SQUARE]

### Sprint (hold)
PC: [SHIFT] / [ALT]
XBOX: [A]
PS2: [X]

### Enter Vehicle
PC: [F] / [ENTER]
XBOX: [Y]
PS2: [TRIANGLE]

### Attack / Fire Weapon
PC: [LMB] / [CTRL] / [NUM LOCK]+[0]
XBOX: [RT]
PS2: [CIRCLE]

### Cycle Weapon / Target (left)
PC: [MMB] / [ENTER]
XBOX: [LEFT]
PS2: [R2]

### Cycle Weapon / Target (right)
PC: [MMB] / [NUM LOCK]+[PERIOD]
XBOX: [RIGHT]
PS2: [L2]

### Center Camera
XBOX: [BLACK]
PS2: [L1]

### Target / Scope View (hold)
PC: [RMB] / [DEL]
XBOX: [LT]
PS2: [R1]

### Look Behind (hold)
PC: [NUM LOCK]+[1] / MMB / CAPS LOCK
XBOX: [RIGHT STICK BUTTON]
PS2: [R3]

## First Person Shooting

### First Person Mode
PC: [RMB] / [DEL]
XBOX: [LT]
PS2: [R1]

### Shoot
PC: [LMB] / [CTRL] / [NUM LOCK]+[0]
XBOX: [RT]
PS2: [CIRCLE]

### Zoom In
PC: [PAGE UP] / [Z]
XBOX: [X]
PS2: [SQUARE]

### Zoom Out
PC: [PAGE DOWN] / [X]
XBOX: [A]
PS2: [X]

### Aim
PC: [MOUSE]
XBOX: [LEFT STICK BUTTON]
PS2: [L3]

### In Vehicle

### Enter / Exit Vehicle
PC: [F] / [ENTER]
XBOX: [Y]
PS2: [TRIANGLE]

### Fire Weapon
PC: [NUM LOCK]+[0] / [CTRL] / [LMB]
XBOX: [B]
PS2: [CIRCLE]

### Hand Brake
PC: [SPACE] / [CTRL] / [RMB]
XBOX: [B]
PS2: [R1]

### Accelerate
PC: [W] / [UP ARROW]
XBOX: [RT]
PS2: [X]

### Brake / Reverse
PC: [S] / [DOWN ARROW]
XBOX: [LT]
PS2: [SQUARE]

### Look Left
PC: [Q] / [NUM LOCK]+[1]
XBOX: [RIGHT STICK BUTTON]
PS2: [L2]

### Look Right
PC: [E] / [NUM LOCK]+[2]
XBOX: [RIGHT STICK BUTTON]
PS2: [R2]

### Look Behind
PC: [NUM LOCK]+[1] + [NUM LOCK]+[2] / [Q]+[E] / [MMB] / [CAPS LOCK]
XBOX: [RT]+[LT]
PS2: [L2]+[R2]

### Steer
PC: [A] / [D] / [LEFT ARROW] / [RIGHT ARROW]
XBOX: [LEFT STICK]
PS2: [LEFT STICK]

### Horn (hold)
PC: [SHIFT] / [ALT]
XBOX: [LEFT STICK BUTTON]
PS2: [L3]

### Cycle Radio Stations (up)
PC: [MMB] / [INSERT] / [R]
XBOX: [BLACK]
PS2: [L1]

### Turret
PC: [NUM LOCK] + [9] / [4] / [6] / [5]
XBOX: [RIGHT STICK]
PS2: [RIGHT STICK]

### Side Mission (Taxi/Ambulance/Vigilante)
PC: [NUMPAD PLUS] / [CAPS LOCK]
XBOX: [X]
PS2: [R3]

## Plane

### Dodo Up / Down
PC: [NUM LOCK] + [9] / [6]
XBOX: [LEFT STICK]

## Miscellaneous

### Begin / End ~30-Second Replay
PC: [F1]

### Save ~30-Second Replay
PC: [F2]

### Load Saved ~30-Second Replay
PC: [F3]
